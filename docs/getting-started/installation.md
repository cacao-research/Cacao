# Installation

## Requirements

- Python 3.7 or higher
- pip package manager

## Install from PyPI

```bash
pip install cacao
```

## Install from Source

```bash
git clone https://github.com/cacao-research/Cacao.git
cd Cacao
pip install -e .
```

## Optional Dependencies

### Desktop Mode

Desktop mode is included by default via `pywebview`. No additional installation needed.

### Documentation

To build the documentation locally:

```bash
pip install cacao[docs]
```

## Verify Installation

After installation, verify Cacao is working:

```bash
cacao --help
```

You should see available CLI commands listed.
